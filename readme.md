# Single Page Application with Laravel 5.5 and Vue.js 2 - VideoIntro

### Frameworks and Libraries:

- Laravel 5.5
- Vue.js 2.2
- Vue Router
- Axios

### PHP Settings Requirement

- Disable `post_max_size` for Uploading of videos purposes

`enable_post_data_reading = On`
`post_max_size = 0`


### Installation

`composer install`

`npm install`
